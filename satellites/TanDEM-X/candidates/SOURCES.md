# TanDEM-X — Candidate Reference Images

**Mission:** TanDEM-X (TerraSAR-X add-on for Digital Elevation Measurement)  
**Agency:** DLR (German Aerospace Center)  
**Band:** X-band SAR  
**Launched:** 21 June 2010  
**Note:** TanDEM-X and TerraSAR-X are near-identical sister satellites flying in close formation. The TerraSAR-X SVG already in this repo is a strong starting point — TanDEM-X has the same bus shape with slight antenna differences.
