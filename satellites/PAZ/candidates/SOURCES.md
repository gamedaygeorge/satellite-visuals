# PAZ — Candidate Reference Images

**Mission:** PAZ (Spanish for "peace")  
**Agency:** Hisdesat / Spanish Ministry of Defence  
**Band:** X-band SAR  
**Launched:** 22 February 2018 (SpaceX Falcon 9)  
**Note:** PAZ is built on the same Airbus DS platform as TerraSAR-X and TanDEM-X. If TerraSAR-X and TanDEM-X SVGs exist in this repo, PAZ is a third member of that family — similar bus, same general shape.

## Notes for SVG Artist

PAZ shares the same Airbus DS SAR platform as TerraSAR-X. The main visual elements are a hexagonal bus body with a large SAR antenna panel and two solar panels. The `TerraSAR-X.svg` is a useful starting reference. Required files:
- `PAZ.svg`
- `PAZ-icon.svg`
- `PAZ-512px.png`
- `PAZ-1024px.png`
